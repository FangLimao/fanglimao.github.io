# 脚本导论

```admonish warning
本章内容需要提前我们了解 JavaScript 基本语法
```

**脚本API（Script API, SAPI）** 是 Minecraft 中的一个脚本系统，可在附加包中实现事件监听与响应、修改世界、操作物品与实体等功能。

SAPI 采用 JavaScript 语言编写，在灵活性上远大于基于 JSON 的数据驱动，但仍无法代替数据驱动提供的注册物品、方块、实体等基本功能。

SAPI 的脚本是独立的，各行为包彼此之间不能互相访问、依赖、读写动态属性。

**看门狗（Watchdog）** 是一个性能监视器，用于监控脚本引擎中运行缓慢的脚本。
